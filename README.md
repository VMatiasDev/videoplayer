<h2 align='center'>Rodando o projeto localmente</h2>

<h3>Pré-Requisitos:</h3>
NodeJs
Yarn

<h3>Instalação:</h3>

1. Clonar o repositório

```sh
   https://github.com/VMatiasDev/videoplayer.git
```

2. Instalar as dependências locais

```sh
   yarn
```

3. Rodar o projeto

```sh
   yarn dev
```
